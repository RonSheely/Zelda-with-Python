# Zelda with Python
A Zelda-like game made with the Pygame library.

![image](https://user-images.githubusercontent.com/85440857/160607397-d085869c-3910-4091-b790-be096ee72b5a.png)

# Controls
| Key | Description |
| :---: | :---: |
| `Arrow Keys` | Move |
| `Space Bar` | Attack |
| `Left Control` | Magic |
| `Q` | Change Weapon |
| `E` | Change Magic |
| `M` | Toggle Upgrade Menu |
